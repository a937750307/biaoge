## 表格裁条生成器
本项目是一个网页版的表格裁条/工资条/成绩单生成工具。

## 演示地址
[https://biaoge.937788.xyz/](https://biaoge.937788.xyz/)

## 使用方法
上传 Excel 或 CSV 文件，或点击上传模板文件体验。
可自定义标题、单位、年月、表头行数和字体。
点击“预览”查看效果，点击“导出为PDF”可打印。

## 使用场景
* 工资条批量生成与打印
* 成绩单、通知单、回执单等明细条制作
* 抽奖券、签到条、礼品领取单等批量票据
* 其他需要将表格内容裁切为单条明细的场合
